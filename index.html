3. Cara Menjalankan & Membuka Akses Publik Karena aplikasi ini bersifat client-side (WebRTC langsung berjalan di browser), Anda tidak butuh setup server VPS yang rumit untuk mencobanya. Namun, WebRTC mewajibkan protokol HTTPS agar fitur Screen Sharing dapat berjalan secara publik. Berikut adalah 2 cara termudah untuk mempublikasikannya secara gratis: Opsi A: Menggunakan GitHub Pages (Sangat Direkomendasikan) Buat repository baru di GitHub Anda. Unggah file index.html ke dalam repository tersebut. Buka tab Settings > Pages di repository Anda. Pada bagian Build and deployment, pilih Branch: main (atau master), lalu klik Save. Dalam beberapa menit, website Anda akan live dengan URL HTTPS gratis (contoh: [https://username.github.io/repo-name/](https://username.github.io/repo-name/)). Opsi B: Menggunakan Netlify / Vercel Drag & drop folder berisi index.html ke halaman Netlify Drop. Netlify akan langsung memberikan tautan HTTPS publik yang bisa diakses siapa saja di internet. Cara Penggunaan: Pengirim (Host): Buka website, klik tombol "Bagikan Layar Saya", pilih jendela/layar yang ingin dibagikan. Salin Room ID yang tertera di pojok kanan atas. Penerima (Viewer): Buka website di perangkat atau lokasi lain, masukkan Room ID dari pengirim ke kolom input, lalu klik "Tonton". Buatkan file jadi dari awal hingga jadi dan di deploy ke vercel melalui github Berikut adalah file kode index.html yang sudah siap jadi, lengkap beserta panduan langkah demi langkah untuk mengunggahnya ke GitHub dan mendeploynya secara gratis ke Vercel agar bisa diakses oleh publik. File Aplikasi Siap Pakai Silakan unduh atau salin file proyek yang telah dibuat berikut ini: Halaman Utama Website: index.html Ikon HTML index HTML Buka Catatan Petunjuk Proyek: README.txt [file-tag: code-generated-file-0-1782841005559365534] Panduan Lengkap: Dari Lokal, GitHub, hingga Deploy Vercel Karena Vercel otomatis menyediakan sertifikat HTTPS, fitur Screen Sharing berbasis WebRTC pada file di atas dipastikan akan langsung berjalan dengan lancar saat diakses publik. Berikut adalah langkah-langkahnya: Langkah 1: Persiapan File di Komputer Anda Buat sebuah folder baru di komputer Anda (misalnya diberi nama public-screenshare). Masukkan file index.html yang telah diunduh dari link di atas ke dalam folder tersebut. Langkah 2: Mengunggah Proyek ke GitHub Ada dua cara untuk mengunggah file ke GitHub: lewat aplikasi browser (paling mudah) atau lewat Git Terminal. Berikut cara termudah menggunakan browser: Buka GitHub dan masuk ke akun Anda. Klik tombol New (atau Create repository) untuk membuat tempat penyimpanan baru. Beri nama repository Anda (misal: screenshare-app), atur menjadi Public, lalu klik Create repository. Di halaman kosong yang muncul, klik tulisan "uploading an existing file". Tarik (drag & drop) file index.html Anda ke dalam kotak yang disediakan di browser. Klik tombol hijau Commit changes di bagian bawah. File Anda sekarang sudah tersimpan aman di GitHub. Langkah 3: Mendeploy ke Vercel (Hanya 1 Menit) Buka situs Vercel dan masuk menggunakan akun GitHub Anda (Sign in with GitHub). Setelah masuk ke Dashboard Vercel, klik tombol Add New... lalu pilih Project. Di bagian Import Repository, Anda akan melihat daftar repository GitHub Anda. Cari repository screenshare-app yang baru dibuat, lalu klik Import. Pada konfigurasi proyek (Configure Project): Framework Preset: Biarkan mendeteksi otomatis atau pilih Other (karena ini hanya file HTML statis biasa). Root Directory & Build Commands: Biarkan kosong / default (tidak perlu diisi apa-apa). Klik tombol Deploy. Tunggu sekitar 10–30 detik hingga proses selesai. Vercel akan menampilkan animasi kembang api dan memberikan tautan domain resmi publik (misalnya: https://screenshare-app.vercel.app). Cara Menggunakannya Setelah Online: Host (Penyiar Layar): Buka link Vercel Anda, klik tombol "Bagikan Layar Saya", lalu pilih layar atau jendela aplikasi yang ingin disiarkan. Salin kode Room ID unik yang muncul di pojok kanan atas. Viewer (Penonton): Berikan link Vercel tersebut beserta Room ID Anda kepada orang lain. Mereka cukup memasukkan Room ID ke kolom yang tersedia, lalu mengklik tombol "Tonton" untuk melihat layar Anda secara langsung (real-time). Gemini adalah AI dan dapat melakukan kesalahan.
